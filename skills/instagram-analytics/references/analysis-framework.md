# Analysis Framework

## 성과 지표 계산

**인게이지먼트율 (Engagement Rate)**
```
ER = (좋아요 + 댓글 + 공유 + 저장) / 도달 수 x 100
```
게시물 유형별(릴스/피드/스토리)로 평균 ER을 계산하고, 벤치마크 기준과 대조하여 평가한다.

**저장율 (Save Rate)**
```
Save Rate = 저장 수 / 도달 수 x 100
```
저장율이 높은 게시물은 알고리즘 도달 확장 가능성이 크다.

**공유율 (Share Rate)**
```
Share Rate = 공유 수 / 도달 수 x 100
```

**팔로워 순증 (Net Growth)**
```
Net Growth = 새 팔로우 - 언팔로우
```

**릴스 시청 완료율** (데이터 있는 경우)
```
Completion Rate = 평균 시청 시간 / 영상 전체 길이 x 100
```

**광고 효율** (광고 게시물만)
```
ROAS는 전환 매출이 있는 경우에만 계산. 없으면 CPM/CPC로 대체.
```

## 콘텐츠 유형별 성과 비교

릴스/피드 포스트/캐러셀/스토리를 비교한다:
- 평균 도달 수
- 평균 인게이지먼트율
- 팔로워 유입 기여도
- 상위 3개 게시물 공통점 분석

**도달과 ER의 교차 패턴을 반드시 확인한다:**
- 도달 증가 + ER 하락: 양적 확산은 됐지만 질적 호응 약화. 광고 트래픽이나 알고리즘 단발 노출 가능성.
- 도달 유지 + ER 상승: 콘텐츠 호응 개선 신호. TOP 3에서 변화 원인을 찾는다.
- 도달 계속 하향: 알고리즘 노출 저하. 포맷 변화를 제안한다.

## 잘하고 있는 방향 (What's Working)

총 반응 수 기준 TOP 5 게시물을 선정하고 공통점을 분석한다:
- 콘텐츠 주제/포맷
- 게시 시간대
- 캡션 스타일
- 썸네일/첫 프레임 유형
- 해시태그 전략
- 저장 수가 높은 게시물 유형

## 개선해야 할 점 (What to Fix)

- ER이 낮은 유형 또는 시기
- 저장/공유 수가 특히 낮은 이유 추론
- 릴스 시청 이탈 가능성
- 콘텐츠 일관성 (주제, 비주얼 톤, 게시 주기)
- 광고 집행 게시물의 유기적 성과 대비 효율
- 팔로워 순증 마이너스 기간: 언팔 급증 시점과 콘텐츠 유형 변화 교차 확인

## 앞으로 공략하면 좋을 것 (Opportunities)

- 저장율이 높은 주제를 더 자주 다룰 것인지
- 릴스 비중 조정
- 콜라보/UGC 유형 시도 가능성
- 스토리를 팔로워 리텐션 도구로 활용
- 광고 집행 게시물 선정 기준 개선
- 게시 빈도 및 요일/시간대 조정

## 광고 효과 심층 분석

광고 집행 게시물이 있는 경우:
- 유기 도달 vs 광고 도달 비율
- 광고 집행 전후 팔로워 반응 비교
- 예산 효율 평가 (CPM 평균: $5~15, CPC 평균: $0.5~3 참고)
- 광고 대상 게시물 선정의 적절성 평가

## 콘텐츠별 마케팅 인사이트 (팀 공유용)

사용자가 별도로 요청하면 아래 계산식으로 콘텐츠별 인사이트를 추가 생성한다:

```
Engagement Rate = (좋아요 + 댓글 + 공유) / 조회수 x 100
좋아요 전환율 = 좋아요 / 조회수 x 100
공유율 = 공유 / 조회수 x 100
```

필수 포함:
- 최근 30일 Top 콘텐츠별 ER 표
- 최근 1주일 포스트 ER 표
- Top 3 / Bottom 3
- Carousel/Post 주제별 인사이트
- College marketing 관점 분석
- 다음 달 콘텐츠 방향
- 팀 공유용 한줄 요약과 우선순위 액션 3개
